What's included

Every release

Manpages from every supported Ubuntu release, from previous LTS releases to the latest development series.

Every repository

Covers main, restricted, universe, and multiverse, not just the packages installed by default.

Structured HTML

Converted from roff source using mandoc, with table of contents, cross-references, and section anchors.

Searchable

Find any command, library, or configuration file by name across all releases.

Always current

Pulled directly from the Ubuntu archive and refreshed regularly to include security and stable updates.

Plain text

Every manpage is also available as a .txt endpoint for scripted consumption.
