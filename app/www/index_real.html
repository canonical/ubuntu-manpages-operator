Welcome

Welcome to the Ubuntu Manpage Repository

This site contains hundreds of thousands of dynamically generated manuals, extracted from every package of every supported version of Ubuntu, and updated on a daily basis. Traditionally, manpages are browsed on the command line. This project renders all such manuals included in Ubuntu into an HTML, web-browsable format.

Other resources

You may also find the following Ubuntu documentation resources useful:

  • User documentation for all Ubuntu operating systems and products
  • Community documentation for participants in Ubuntu communities
  • Project documentation for contributors to Ubuntu development
  • Developer documentation for using Ubuntu as a development platform

Navigating the repository

  • Browse the repository as a hierarchical file structure
  • Use the search form at the top of every page
  • Download the 'dman' shell script to run on a command line
